## Project Description
This project is a responsive landing page designed for a startup company offering digital services. It showcases a modern business website structure using HTML, CSS and JavaScript.

The website includes:

- Hero Section
- About Company Section
- Services Section
- Client Testimonials
- Contact Section
- Responsive Navigation Bar
- Footer

## How to Run
### Using VS Code
1. Open project folder in VS Code  
2. Install **Live Server** extension  
3. Right click `index.html`  
4. Select **Open with Live Server**

### Or
Open `index.html` directly in browser.
